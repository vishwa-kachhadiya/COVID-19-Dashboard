# 🦠 COVID-19 Dashboard using Tableau

## 📌 Overview

This project presents an interactive COVID-19 dashboard developed using Tableau to visualize and analyze global pandemic statistics. The dashboard provides insights into confirmed cases, deaths, recoveries, and geographical distribution through interactive visualizations and filters.

The dashboard enables users to explore COVID-19 trends across different countries and better understand the impact of the pandemic using intuitive visual analytics.

---

## 📊 Dashboard Features

- 🌍 Interactive world map showing COVID-19 cases by country
- 📈 Trend analysis of confirmed cases over time
- 🟢 KPI indicators for total confirmed cases
- 🔴 KPI indicators for total death cases
- 📍 Bubble map representing geographical distribution
- 🎛 Interactive filters for data exploration
- 📊 Easy-to-understand visual analytics
